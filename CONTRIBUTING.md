# Contributing to WhatsApp AI Agent Bot

Thank you for your interest in contributing! We welcome all contributions, no matter how small.

## How to Contribute

### Reporting Bugs
- Check if the bug already exists in [Issues](https://github.com/Sudeep5363/Whatsapp_AI_Agent/issues)
- Create a detailed bug report with:
  - Python version and OS
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots/error logs if applicable

### Feature Requests
- Suggest new features via [Issues](https://github.com/Sudeep5363/Whatsapp_AI_Agent/issues)
- Clearly describe the use case and expected behavior

### Code Contributions
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/YourFeature`
3. Make your changes and test thoroughly
4. Add docstrings and comments for clarity
5. Commit with clear messages: `git commit -m 'Add: YourFeature'`
6. Push to branch: `git push origin feature/YourFeature`
7. Open a Pull Request with a detailed description

## Code Style Guidelines

- Use **PEP 8** style guide
- Add docstrings to all functions
- Use type hints where possible
- Keep lines under 100 characters
- Use meaningful variable names

## Testing

Before submitting:
- Test with Python 3.8+
- Test on Windows/Linux/macOS if possible
- Verify no new runtime errors
- Check Chrome compatibility

## Pull Request Guidelines

- One feature per PR
- Update README if needed
- Add meaningful commit messages
- Keep PRs focused and small
- Be open to feedback

## Questions?

Feel free to open an issue or contact the maintainer.
